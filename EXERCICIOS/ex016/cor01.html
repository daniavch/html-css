Representação por Código Hexadecimal

Toda cor é formada por um conjunto de combinações de vermelho, azul e verde. 0 é o menor falor e F o maior no parâmetro hexadecimal.

A partir do padrão RGB podemos entender o código hexadecimal da seguinte forma: #00(determina o vermelho, RED)00(determina o verde, GREEN)ff(determina o azul, BLUE). No código #0000ff temos valor 0 de vermelho, valor 0 de verde e o maior valor de azul.

Notação RGB

0 é o valor nulo na notação RGB e 255 o valor máximo. Segue a mesma lógica do código hexadecimal. Primeiro número: quantidade de vermelho, segundo número: quantidade de verde, terceiro número: quantidade de azul. Entendendo RGB você entende o uso de cores em qualquer aparelho que use uma tela.

Notação HSL

Outra maneira de descrever cores é usando a notação hsl. HSL é uma sigla para HUE, SATURATION e LIGHTNESS. Como podemos imaginar, nesse padrão usamos matiz, saturação e luminosidade (características de todas as cores) para identificar cada cor.
