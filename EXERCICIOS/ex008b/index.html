Outras formatações

Código-fonte

O código-fonte é o código original de alguma programação.
O comando documento.getElementById('teste') é escrito em linguagem JavaScript.
A tag CODE serve para indicar que o texto é um código e deve ser formatado em monoespaçamento. Como podemos ver no exemplo de JS.
A tag PRE serve para indicar que a formatação usada no código HTML deve ser mantida na visualização da página. Quando queremos apresentar códigos devemos usar essa tag para manter a indentação, espaços, quebras de linhas, etc.

Citações

Citações podem ser marcadas no HTML com a tag Q ou com a BLOCKQUOTE. A tag Q deixa o texto entre aspas, é usada geralmente no meio do parágrafo. Este é um exemplo de uso da tag Q.
A tag BLOCKQUOTE nos permite criar citações mais longas, que são inseridas em blocos. Usando o parâmetro CITE podemos colocar o autor da citação. Esse parâmetro não fica visível na página mas é importante para os mecanismos de busca. Exemplo:
Segundo Jeff Noble:

A diferença entre elementos inline e um bloco de texto é importante.

Abreviações
