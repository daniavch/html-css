Trabalhando com Listas

Listas Ordenadas

A tag OL marca uma lista ordenada, ou seja, com uma ordem lógica. Os itens da lista devem ser marcados com a tag LI

  1. Acordar
  2. Tomar café
  3. Escovar os dentes

As listas ordenadas não precisam ser iniciadas apenas por números. Ao usar o parâmetro TYPE podemos decicidir entre algarismos comuns ou romanos, letras maiúsculas ou minúsculas.

  1. Estudar
  2. Comer
  3. Correr

Você também pode indicar o início da contagem usando o parâmetro start. Por exemplo, o parâmetro type=“I” start = “5” vai gerar itens numerados como V, VI, VII, VIII, IX,

  1. Trigo
  2. Pão
  3. Queijo
  4. Tomate

Listas não ordenadas

Listas não ordenadas são aquelas que não apresentam uma sequência lógica determinada. É apenas uma listagem de itens em ordem aleatória. A tag para esse tipo de lista é UL e os itens dentro dela usam a mesma tag de OL, ou seja, LI. Ao usar o parâmetro TYPE podemos mudar o símbolo que aparece antes dos itens da lista. As opções são disc, circle e square.

  • Trigo
  • Pão
  • Queijo
  • Tomate
